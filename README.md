# HydroDome Interactive Maps

Interactive wildfire risk and WUI (Wildland-Urban Interface) maps for HydroDome by Carmanah Wildfire.

## Maps

- **[Canadian WUI Infrastructure](https://tetontopo.github.io/hydrodome-maps/)** - Communities at risk with historic fire perimeters
- **[BC WUI Communities](https://tetontopo.github.io/hydrodome-maps/bc-wui-communities.html)** - British Columbia at-risk communities
- **[Fire Perimeters - North America](https://tetontopo.github.io/hydrodome-maps/fire-perimeters-north-america.html)** - Continental fire history

Built with Leaflet.js | Data: NRCan CWFIS, Stats Canada 2021
